# Center.jquery
Center any element vartically and horizontally with a cute animation. This is an example jQuery Plugin for future referrence.

##### How to activate?
`$('.center-image').center();`

##### Default options
* vartical: true 
* horizontal: true
* animate: false

##### Activate with options
```
$('.center-image').center({
	vartical: false,
	animate: true
});
```
